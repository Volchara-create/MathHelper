# MathHelper

Персональний онлайн-помічник з математики для школярів 1-11 класів.

**Живий сайт:** [rostyslavv.vibe.brobots.org.ua](https://rostyslavv.vibe.brobots.org.ua)

## Про проект

MathHelper — освітній веб-додаток з персоналізованим контентом по класу. Учень реєструється, обирає клас — і бачить лише те що йому потрібно: формули, таблиці, підручники.

## Функціонал

- **Реєстрація та вхід** — звичайний або через Google акаунт
- **Особистий кабінет** — формули та матеріали по класу (1-11 кл.)
- **Формули** — розбиті по класах:
  - 1-2 кл.: Додавання, Віднімання з прикладами 🍎🍇
  - 3-4 кл.: Множення, Ділення, Геометрія, Дроби
  - 5-7 кл.: Відсотки, Степені, Скорочене множення
  - 8-9 кл.: Квадратне рівняння, Прогресії, Тригонометрія
  - 10-11 кл.: Логарифми, Похідна, Інтеграл
- **Таблиці** — множення, квадрати, sin/cos/tan (по класах)
- **Геометрія** — SVG-ілюстрації з формулами фігур
- **Підручники** — посилання на безкоштовні підручники МОН 1-11 кл.
- **Конспекти** — особисті нотатки з збереженням у базі даних
- **Зміна класу** — можна змінити клас у будь-який момент

## Технології

- Frontend: HTML, CSS, JavaScript (без фреймворків)
- Backend: Node.js + Express
- База даних: PostgreSQL + Prisma ORM
- Авторизація: JWT + Google OAuth 2.0
- Деплой: Docker + VPS

## Структура

```
MathHelper/
├── index.html        — розмітка
├── style.css         — стилі
├── script.js         — логіка формул, таблиць, графіків
├── auth.js           — авторизація, дашборд, конспекти
├── backend/
│   ├── server.js     — Express API
│   ├── prisma/       — схема БД
│   └── .env          — змінні середовища
├── Dockerfile
└── docker-compose.yml
```

## Запуск локально

```bash
cd backend
npm install
npx prisma migrate dev
node server.js
```

Сайт доступний на http://localhost:3001

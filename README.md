# MathHelper

Інтерактивний онлайн-помічник з математики для школярів та студентів.

## Про проект

MathHelper — це безкоштовний веб-додаток для вивчення математики та геометрії. Містить формули, інтерактивний робочий стіл, тренувальні задачі та калькулятор — все в одному місці.

## Функціонал

- **Формули геометрії** — понад 30 формул для плоских і об'ємних фігур з інтерактивними SVG-ілюстраціями:
  - Плоскі фігури: прямокутник, ромб, паралелограм, трапеція, коло, трикутник
  - Об'ємні фігури: куб, паралелепіпед, циліндр, конус, піраміда
- **Робочий стіл** — координатна площина з інструментами:
  - Побудова графіків математичних функцій
  - Розміщення геометричних фігур (коло, прямокутник, квадрат, трикутник, ромб, трапеція)
  - Нотатки прямо на полотні
  - Вбудований калькулятор з тригонометричними функціями
  - Масштабування та переміщення мишею
- **Тренувальні задачі** — інтерактивні задачі з геометрії з перевіркою відповіді та покроковим розв'язанням

## Структура файлів

```
MathHelper/
├── index.html    — HTML-розмітка сторінки
├── style.css     — всі стилі (CSS)
└── script.js     — вся логіка (JavaScript)
```

## Як запустити

Просто відкрий `index.html` у браузері — або скористайся будь-яким локальним сервером:

```bash
npx serve .
```

Або перейди на живу версію: [volchara-create.github.io/MathHelper](https://volchara-create.github.io/MathHelper/)

## Технології

- Чистий HTML, CSS, JavaScript (без фреймворків)
- Canvas API для координатної площини
- Google Fonts (Roboto, Orbitron, Fira Code)

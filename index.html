📋 Основні формули

Основна тотожність
sin²α + cos²α = 1
tan і ctg
tan α = sin α / cos α
ctg α = cos α / sin α
tan α · ctg α = 1
Радіани
180° = π рад
1° = π/180 рад
Формули суми кутів
sin(α±β) = sinα·cosβ ± cosα·sinβ
cos(α±β) = cosα·cosβ ∓ sinα·sinβ
Подвійний кут
sin 2α = 2·sinα·cosα
cos 2α = cos²α − sin²α
Пониження степеня
sin²α = (1 − cos 2α) / 2
cos²α = (1 + cos 2α) / 2
tan²α = (1 − cos 2α) / (1 + cos 2α)
Формули зведення
sin(90°−α) = cos α
cos(90°−α) = sin α
sin(180°−α) = sin α
cos(180°−α) = −cos α
sin(−α) = −sin α
cos(−α) = cos α
Теорема синусів / косинусів
a/sin A = b/sin B = c/sin C
c² = a² + b² − 2ab·cos C

Про MathHelper

MathHelper — це потужний інструмент для вивчення математики та геометрії. Додаток містить понад 30 формул для різних геометричних фігур та можливість побудови графіків функцій.

Можливості:

📐 Формули для плоских фігур: прямокутник, ромб, паралелограм, трапеція, коло, трикутник

📦 Формули для об'ємних фігур: куб, паралелепіпед, циліндр, конус, піраміда

📈 Побудова графіків математичних функцій на координатній площині

🔷 Геометричні фігури прямо на координатній площині

🎯 Інтерактивні візуалізації кожної формули

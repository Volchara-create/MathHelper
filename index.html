📋 Основні формули

Основна тотожність
sin²α + cos²α = 1
tan і ctg
tan α = sin α / cos α
ctg α = cos α / sin α
tan α · ctg α = 1
Радіани
180° = π рад
1° = π/180 рад
Формули суми кутів
sin(α±β) = sinα·cosβ ± cosα·sinβ
cos(α±β) = cosα·cosβ ∓ sinα·sinβ
Подвійний кут
sin 2α = 2·sinα·cosα
cos 2α = cos²α − sin²α
Пониження степеня
sin²α = (1 − cos 2α) / 2
cos²α = (1 + cos 2α) / 2
tan²α = (1 − cos 2α) / (1 + cos 2α)
Формули зведення
sin(90°−α) = cos α
cos(90°−α) = sin α
sin(180°−α) = sin α
cos(180°−α) = −cos α
sin(−α) = −sin α
cos(−α) = cos α
Теорема синусів / косинусів
a/sin A = b/sin B = c/sin C
c² = a² + b² − 2ab·cos C

📚 Формули

Степені та коріні
Степіньaⁿ · aᵐ = aⁿ⁺ᵐ
Діленняaⁿ / aᵐ = aⁿ⁻ᵐ
Степінь степеня(aⁿ)ᵐ = aⁿ·ᵐ
Добуток(a·b)ⁿ = aⁿ·bⁿ
Корінь√(a·b) = √a·√b
Нульовий степіньa⁰ = 1
Скорочене множення
Квадрат суми(a+b)² = a²+2ab+b²
Квадрат різниці(a−b)² = a²−2ab+b²
Різниця квадратівa²−b² = (a+b)(a−b)
Куб суми(a+b)³ = a³+3a²b+3ab²+b³
Куб різниці(a−b)³ = a³−3a²b+3ab²−b³
Сума кубівa³+b³ = (a+b)(a²−ab+b²)
Квадратне рівняння
Загальний виглядax² + bx + c = 0
ДискримінантD = b² − 4ac
Кореніx = (−b ± √D) / 2a
D > 0два корені
D = 0один корінь x = −b/2a
D < 0немає коренів
Лінійна функція
Формулаy = kx + b
k > 0функція зростає
k < 0функція спадає
bзсув по осі y
k = 0y = b (горизонталь)
Прогресії
Арифм. nй членaₙ = a₁ + (n−1)·d
Сума n членівSₙ = n(a₁+aₙ)/2
Геом. nй членbₙ = b₁ · qⁿ⁻¹
Сума геом.Sₙ = b₁(qⁿ−1)/(q−1)
Логарифми
Означенняlogₐ(b) = x ↔ aˣ = b
Добутокlogₐ(xy) = logₐx + logₐy
Часткаlogₐ(x/y) = logₐx − logₐy
Степіньlogₐ(xⁿ) = n·logₐx
Натуральнийln(x) = logₑ(x)
Похідна (10-11 кл.)
Константа(C)' = 0
Степінь(xⁿ)' = n·xⁿ⁻¹
Сума(u+v)' = u' + v'
Добуток(uv)' = u'v + uv'
sin(sin x)' = cos x
cos(cos x)' = −sin x
Інтеграл (11 кл.)
Степінь∫xⁿdx = xⁿ⁺¹/(n+1) + C
sin∫sin x dx = −cos x + C
cos∫cos x dx = sin x + C
1/x∫(1/x)dx = ln|x| + C
eˣ∫eˣdx = eˣ + C

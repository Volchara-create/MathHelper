# Аудит проекту MathHelper
_Дата: 11.04.2026_

---

## 🔴 КРИТИЧНІ — виправити першочергово

1. **`nbSaveAsNote()` — зламано на GitHub Pages** (`script.js:2045`)
   Використовує `/notes` замість `${API}/notes`. Кнопка "Зберегти конспект" у зошиті не працює на GitHub Pages.

2. **`tasksInit()` — показує задачі 7 класу незалогіненим** (`script.js:1621`)
   `getUserGrade()` повертає `null` → завантажуються задачі для 7 класу без попередження. Немає повідомлення "увійди щоб бачити задачі свого класу".

3. **Header ламається на мобільних < 400px** (`style.css`)
   Лого + dark-toggle + 3 кнопки auth не вміщуються. Немає `@media` для header. Кнопки можуть обрізатися.

---

## 🟡 СЕРЕДНІ — виправити найближчим часом

4. **`buildGeoTab()` не оновлюється при зміні класу** (`script.js:526`)
   Guard `if (grid.children.length > 0) return` не дає оновитися після зміни класу в налаштуваннях. Геометрія показує старі фігури.

5. **Loading-стан для нотаток на дашборді відсутній** (`auth.js:607`)
   `dash-recent-notes` порожній поки завантажується — немає "Завантаження...". В drawer є, на дашборді — ні. Непослідовний UX.

6. **Подвійний запит при логіні/реєстрації** (`auth.js:348`)
   Немає `btn.disabled = true` під час fetch. При подвійному кліку — два запити.

7. **`startTrial()` використовує `alert()`** (`auth.js:481`)
   Нативний alert блокує UI. Потрібна модалка або toast.

8. **Quick-menu перекриває контент на мобільних** (`style.css:866`)
   `padding-bottom:72px` застосовується до `<main>`, але секції не завжди прокручуються — кнопки внизу можуть бути недоступні.

9. **Граф (workspace) не працює на touch-екранах** (`script.js`)
   Тільки mouse-events (mousedown/mousemove). Немає touchstart/touchmove — пан і зум недоступні на мобільних.

## 🟢 МАЛІ — косметика і рефакторинг

10. **Dark mode CSS — частина карток без темної теми** (`style.css`)
    `.dark .dash-formula-card`, `.dark .alg-card` — неправильний селектор (`.dark` замість `body.dark`). Ці елементи не отримують темні стилі.

11. **`confirmGoogleGrade()` — XSS-ризик в onclick** (`auth.js:120`)
    JWT передається як рядок в HTML атрибут. Якщо містить `'` — зламається. Потрібен data-атрибут.

12. **`settingsSaveGrade()` — зайвий setTimeout після closeSettings** (`auth.js:463`)
    `finally` завжди виконується навіть після успішного закриття модалки — зайвий таймер.

13. **Service Worker не очищує старий кеш при оновленні** (`sw.js`)
    Відсутня логіка версіонування кешу. При деплої користувачі можуть бачити старі файли.

14. **`updateNavForGrade()` вмикає dark mode при першому логіні** (`script.js:1658`)
    Примусово додає `body.dark` якщо `mathhelper-dark` не встановлений — може дивувати нових юзерів.

15. **Trig sticky panel не працює на мобільних** (`style.css:394`)
    `position:sticky` в flex-колонці на мобільних → `position:static`. Формули прокручуються.

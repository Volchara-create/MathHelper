# MathHelper — Roadmap і план покращень

## Поточний стан (квітень 2026)

### Реалізовано
- Реєстрація / вхід (email + Google OAuth)
- Особистий кабінет з вибором класу (1-11)
- Формули по класах з фільтрацією (Алгебра, Геометрія, Тригонометрія, Таблиці)
- Геометрія — SVG-фігури з формулами та модальними вікнами
- Підручники МОН (безкоштовні, по класу)
- Конспекти (особисті нотатки у БД)
- Квіз — питання по класах 7-11 з поясненнями, відстеження помилок
- Квіз — продовження незавершеного тесту (зберігається прогрес)
- Квіз — розбір тільки помилок із навігацією Попереднє/Наступне
- Квіз — статистика праворуч від тем: три вкладки (Тиждень / Місяць / Весь час)
- Квіз — статистика постійна (без скидання), батьки бачать реальний прогрес
- Квіз — кожна відповідь зберігається з датою → можливий звіт за будь-який період
- Квіз — % правильних на картках тем (зелений/жовтий/червоний)
- НМТ симулятор — повний тест з таймером, кнопка "Записати ✓", заголовок по центру
- Зошит і Калькулятор як бічні панелі (поруч з будь-яким блоком)
- Навігація: кнопки ← і 🏠 Головна у шапці на всіх сторінках
- Швидке меню: Калькулятор, Зошит, НМТ, ШІ, Pro (без дублювання "Назад")

## Наступні покращення (пріоритет: ↑ вище = важливіше)

### 🔴 Високий пріоритет

- [ ] **Формули — клік для збільшення + доведення**
  Натиснув на картку формули в Алгебрі → відкривається модалка з великим записом,
  поясненням "звідки береться" і прикладом застосування.
  Аналог: в Геометрії вже є такі модалки — зробити так само для Алгебри/Тригонометрії.

- [ ] **Квіз — більше питань для 9-11 класу**
  Геометрія 9-10 (кола, вписані/описані, вектори), стереометрія 11 класу,
  комбінаторика, теорія ймовірностей.

- [ ] **Формули — стереометрія для 11 класу**
  Секція Геометрія не має призми, піраміди, зрізаної піраміди — для НМТ обов'язково.

### 🟡 Середній пріоритет

- [ ] **Місячний підсумок статистики**
  В кінці місяця автоматично зберігати знімок результатів по кожній темі.
  Показувати порівняння: "Квітень: 65% → Травень: 78% ⬆️".
  Технічно: при першому відкритті нового місяця — зберегти `mh_quiz_monthly_YYYY-MM`.

- [ ] **Щоденний стрік (streak)**
  Показувати скільки днів поспіль учень займається (🔥 5 днів).
  Зберігати в localStorage + БД для авторизованих.

- [ ] **Щоденна норма**
  Учень або батько встановлює "5 питань на день".
  Після виконання — зелена галочка на день у тижневому графіку активності.

- [ ] **Квіз — адаптивна складність**
  Якщо тема пройдена без помилок 3 рази — помічати зеленим ✅ "Засвоєно".
  Якщо % нижче 60% — пропонувати переглянути формули перед тестом.

- [ ] **Картки тем у квізі — кількість питань**
  Показувати "8 питань" під назвою теми, щоб учень знав що чекати.

### 🟢 Низький пріоритет / Майбутнє

- [ ] **PRO: ШІ-помічник**
  Учень вводить умову задачі → Claude API розвʼязує покроково українською.
  ~$0.001 за запит, тільки PRO.

- [ ] **PRO: Покроковий розв'язок у квізі**
  Після неправильної відповіді — детальний розбір з кроками (зараз тільки пояснення).

- [ ] **PRO: Завантаження конспекту як PDF**

- [ ] **Профіль батьків**
  Батько бачить статистику дитини, може встановити щоденну норму і отримувати
  сповіщення (email/Telegram) якщо дитина не займалась більше 2 днів.

## Технічні деталі статистики

### Формат збереження (localStorage)
```
mh_quiz_stats_v2: {
  "2026-04-12": { algebra: {total:5, wrong:2}, geometry: {total:3, wrong:1} },
  "2026-04-13": { equations: {total:8, wrong:3} },
  ...
}
```
- Кожна відповідь зберігається з точною датою
- Вкладки Тиждень/Місяць/Весь час рахуються з одних даних
- Без можливості скидання — батьки бачать чесну картину

### Майбутній місячний архів
```
mh_quiz_monthly_2026-04: { algebra: {total:45, wrong:12, pct:73}, ... }
```
Зберігається автоматично при першому запуску нового місяця.

## Примітки

- Формули, таблиці, підручники — **ЗАВЖДИ безкоштовно**. Це основна цінність сайту.
- Монетизація на зручності (нотатки, AI) — не на знаннях.
- Цільова аудиторія: **батьки реєструють, діти займаються**.
  Мотивація батьків = контроль прогресу. Мотивація дітей = стріки і досягнення.
- Статистика без скидання — принципове рішення для довіри батьків.
